## Essential environment variables and files:
Add .env file with this variables and fill them as needed:
```
DATA_BASE=
USERS_COLLECTION=
POSTS_COLLECTION=
COMMENTS_COLLECTION=
VOTES_COLLECTION=

ACCESS_TOKEN_NAME=
ACCESS_TOKEN_SECRET=
ACCESS_TOKEN_TTL=

REFRESH_TOKEN_SECRET=
REFRESH_TOKEN_TT=

PORT=
CLIENT_DNS=
```
Add ssl folder and add inside:
- certificate.pem
- key.pem
